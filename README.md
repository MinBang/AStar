# AStar

== Manual ==<br>
===================================================<br>
LButtonDown(Mouse) - set start point<br>
LButtonDown(Mouse) (after set start) - set obstacle<br>
RButtonDown(Mouse) - set goal<br>
SpaceBar - run<br>
EnterKey - reset<br>
